**SuvidhaHub – Full-Stack Cyber Café / Jan Seva Kendra Management App | React + Tailwind CSS**

In this video, I walk through **SuvidhaHub** — a complete government services portal built for cyber cafés and Jan Seva Kendras (Common Service Centres). The app lets citizens apply for official government documents online, while vendors (retailers) manage applications and track commissions, all overseen by a central admin.

🔧 **Tech Stack:**
- React 18 + Vite
- Tailwind CSS
- React Router DOM v6
- Recharts
- Lucide React Icons

👥 **Three Role System:**
- **Admin** – Full back-office: manage vendors, customers, applications, payments, reports & notifications
- **Vendor** – Retailer dashboard: handle assigned applications, track commission wallet & earnings
- **Customer** – Browse 30+ government services, apply online, and track application status

📋 **Services Covered:**
Aadhaar, PAN, Passport, Driving Licence, Voter ID, Ration Card, Birth/Death Certificates, Income/Caste/Domicile Certificates, Kisan Registration, Electricity Bill, JanAadhar, and more.

⚡ Built as a fully functional prototype with mock data — perfect as a reference project for freelancers, developers building CSC software, or anyone interested in multi-role React SPA architecture.
